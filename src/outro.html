Books

As a general introduction to Python programming which focuses on linguistic applications, I‚Äôve already recommended Natural Language Processing with Python: Analyzing Text with the Natural Language Toolkit by Steven Bird, Ewan Klein and Edward Loper, and I‚Äôm going to recommend it again. It‚Äôs a great resource, all the more useful since it‚Äôs freely available online. Depending on what your immediate needs are, this may be a strength or a weakness, but in the long run, I would argue that every programming linguist should spend some time honing their programming skills instead of always blindly following how-to style recipes, because even copy-pasting black box code can go seriously wrong if you don‚Äôt have a larger picture of what‚Äôs going on.

On perhaps a more practical note, I can definitely recommend Jake VanderPlas‚Äôs Python Data Science Handbook, an other great resource which is also freely available online. This has traditionally been the domain of R, especially among linguists, but R is a very idiosyncratic language which encourages the copy-paste, black box approach: while it sometimes provides pleasant and easy-to-use abstractions (especially in the tidyverse third-party packages), building them yourself or wiring them together can be challenging because the underlying language is not really well-designed, edge cases and surprising behaviors abound. Python is much easier to wrap your head around, perhaps because it has always been intended as a more general-purpose programming language, but by the same token, it can be sometimes hard to know which libraries and techniques to use when getting started with data analysis in Python. The Python Data Science Handbook is there to help you with that.
